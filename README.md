# Calculator Web App 

This is a simple **Calculator Web Application** developed using **HTML, CSS, and JavaScript**.
It allows users to perform basic arithmetic calculations easily.

 Features

* Performs addition, subtraction, multiplication, and division
* Clear (`C`) button to reset the display
* Simple and clean user interface
 Technologies Used

* HTML
* CSS
* JavaScript

## Project Files

* `calculator.html` – Calculator structure
* `calculator.css` – Styling and layout

## How to Run

Open `calculator.html` in any web browser.
